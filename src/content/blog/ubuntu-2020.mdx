# My Experience with Ubuntu in 2020

**Author:** hungsu | **Dates:** Installed 2020-11-10 → Retired 2021-01-01 | **OS:** Ubuntu 18.04

---

## ✅ Things I Liked (Will Miss)

| # | Feature | Details |
|---|---------|---------|
| 1 | **Window Management** | GNOME + **PaperWM** eliminates need for multiple monitors; calls macOS/Windows floating windows "prehistoric" — plans to build similar for Windows |
| 2 | **USB Boot** | Runs entirely off USB drive; Windows refuses USB installation |
| 3 | **Update Control** | No forced updates; fully user-controlled |
| 4 | **No Bloatware** | No Cortana, Candy Crush, or pre-installed junk |
| 5 | **Fewer Notifications** | Windows nags: Edge, OneDrive, virus scans — Linux doesn't |
| 6 | **Dev Performance** | Builds and fetches run faster |

---

## ⚠️ Annoyances (Not Deal-Breakers)

| Issue | Details | Resolution |
|-------|---------|------------|
| **Audio** | DualShock 4 3.5mm jack not detected; PulseAudio Volume Control fixed output but added game audio delay | Config tweak required |
| **Gamepad** | DualShock 4 fussy in games; worked for others out of the box | Shell commands + Googling |
| **App Install Location** | Windows: `Program Files` (C:), macOS: `Applications` → Linux best practice: **`/opt`** | [AskUbuntu reference](https://askubuntu.com/questions/1148/when-installing-user-applications-where-do-best-practices-suggest-they-be-loc) |
| **F13/F14 Keys** | Registered as `UNK191` (Tools) and `UNK192` | — |
| **Screenshots** | GNOME tool bindable to `Alt`+`Shift`+`$` | Works after fiddling |
| **App Formats** | Still unclear: **snap vs. AppImage vs. Flatpak** | — |

## ❌ Deal-Breakers (Forced Return to Windows)

### Software & Games That Don't Work
- **AirExplorer** — no Linux substitute
- **Adobe Suite** — some success via [WinApps](https://github.com/Fmstrat/winapps) but not reliable
- **Games** (non-exhaustive):
  - Destiny 2 (GeForce NOW = too much latency)
  - EA Games (Star Wars Fallen Order)
  - Epic exclusives (Fortnite, The Pathless)
  - Vermintide 2
  - The Witcher 3 (GeForce NOW = lag)
- **Benchmarking**: Cinebench, CrystalDiskMark, UserBenchmark
- **Accessibility**: NVDA screen reader, Internet Explorer
- **VR**: Windows Mixed Reality headset — "even more difficult"
- **Keyboard Config**: GK68XS programming app Windows-only

### UX/Workflow Failures
| Issue | Impact |
|-------|--------|
| **Emoji Picker** | GNOME picker only works in GTK apps; fails in web/Electron (Twitter, Slack, Discord) |
| **Screen Sharing** | All monitors shared as single display → removed extra monitors |
| **Cyberpunk 2077** | [ProtonDB](https://www.protondb.com/app/1091500): lower FPS, more crashes on Linux |
| **Color Reproduction** | "Much much worse" on same monitor; blues/purples most affected |
| **PWAs** | YouTube Music runs as Chromium tab only; Windows: `Win` → type "yout" → seamless launch |
| **Sleep/Suspend** | Linux unable to sleep — cold boots only |

## 🏁 Verdict

> **"I cannot use Linux as my primary OS. But I will very happily use Linux on my home server that doesn't need games or media playback."**

## Key Takeaways for Future Linux Attempts

1. **Window management** (PaperWM) is a killer feature worth porting
2. **Gaming + anti-cheat + VR** remain the hardest blockers
3. **Color management** and **audio latency** need investigation
4. **PWA/desktop integration** for web apps is lacking
5. **Server/headless use case** = ideal Linux fit
